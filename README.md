## 📌 Descrição
Projeto demonstrando o uso de partículas na Unity em três cenários:
- 🌧️ Chuva  
- 🎆 Fogos de artifício  
- 🔥 Mini jogo com fogo  

## ⚙️ Tecnologias
- Unity  
- C#  

## 🎯 Objetivo
Explorar efeitos visuais e interação usando Particle System.

## ▶️ Execução
Baixar arquivos "GAME.zip", "ParticleSystem.zip" e "Rain-main.zip", extraí-los e executar os arquivos que contém ".exe".
